New text-to-speech

Date: 2024-01-15

In first version I used pyaudio for coverting speech to text, however it wasn't quite realistic. That's why I decided to use Eleven Labs. It will work perfectly in my case, it's multilanguage and generates high quallity speech. 

class TTSService:
def __init__(self, api_key):
    self.client = ElevenLabs(api_key=api_key)
    self.voice = "PmQgD9hinEVn8YJXrTGy"
    self.model = "eleven_multilingual_v2"
                    
def play(self, text):
    audio = self.client.generate(
        text=text,
        voice=self.voice,
        model=self.model
    )
    play(audio)

With this code I can simply pass phrase to the play function.

The only thing is that Eleven Labs process input on server and it works with delay, however I think quallity worth it.
