New speech-to-text

Date: 2024-01-15

When I first started building my voice assistant, I used Google Speech Recognition for converting speech to text. However it wasn't fast enough. The main issue was its reliance on an internet connection. Every time a command was spoken, the audio had to be sent to Google's servers for processing, which caused a huge delay.

To improve performance, I switched to Vosk, an open-source speech recognition system that runs locally. By processing everything directly on the machine, Vosk doesn't nrrd the internet connection what significantly reduces the delay.

def start_listening(self):
    if not self.is_listening:
        self.is_listening = True
        self.recognized_text = None
        self.thread = threading.Thread(target=self._listen)
        self.thread.start()
        self.thread.join()  # Wait for the thread to finish
        return self.recognized_text
                
def stop_listening(self):
    if self.is_listening:
        self.is_listening = False
        if self.thread is not None:
            self.thread.join()
            self.thread = None
